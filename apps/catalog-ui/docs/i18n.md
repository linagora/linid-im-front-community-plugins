# Catalog UI - Translation Keys Documentation

This document lists the translation keys used by the catalog-ui shared components. Scopes are provided by the host application; ensure the keys below exist in your locale files.

## BaseLayout

**File:** apps/catalog-ui/src/layouts/BaseLayout.vue

| Key                   | Description            | Usage                | Parameters |
| --------------------- | ---------------------- | -------------------- | ---------- |
| `application.title`   | Application title text | Header toolbar title | -          |
| `application.version` | Version badge text     | Header badge label   | -          |

**Example:**

```json
{
  "application": {
    "title": "LinID Identity Manager",
    "version": "v1.0.0"
  }
}
```

## ButtonsCard

**File:** apps/catalog-ui/src/components/card/ButtonsCard.vue

| Key                                        | Description                  | Usage                 | Parameters |
| ------------------------------------------ | ---------------------------- | --------------------- | ---------- |
| `[INSTANCE_ID].ButtonsCard.title`          | Section title (optional)     | Card header           | -          |
| `[INSTANCE_ID].ButtonsCard.cancel`         | Cancel button label          | Cancel button text    | -          |
| `[INSTANCE_ID].ButtonsCard.confirm`        | Confirm button label         | Confirm button text   | -          |
| `[INSTANCE_ID].ButtonsCard.confirmLoading` | Confirm button loading label | Confirm button loader | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "ButtonsCard": {
      "title": "Actions",
      "cancel": "Cancel",
      "confirm": "Save",
      "confirmLoading": "Saving..."
    }
  }
}
```

## AdvancedSearchCard

**File:** apps/catalog-ui/src/components/card/AdvancedSearchCard.vue

| Key                                            | Description                        | Usage               | Parameters |
| ---------------------------------------------- | ---------------------------------- | ------------------- | ---------- |
| `[INSTANCE_ID].AdvancedSearchCard.title`       | Card title (optional)              | Card header         | -          |
| `[INSTANCE_ID].AdvancedSearchCard.moreFilters` | Label to expand advanced filters   | Toggle button label | -          |
| `[INSTANCE_ID].AdvancedSearchCard.lessFilters` | Label to collapse advanced filters | Toggle button label | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "AdvancedSearchCard": {
      "title": "Filters",
      "moreFilters": "More filters",
      "lessFilters": "Fewer filters"
    }
  }
}
```

## EntityDetailsCard

**File:** apps/catalog-ui/src/components/card/EntityDetailsCard.vue

| Key                                                       | Description                        | Usage                 | Parameters             |
| --------------------------------------------------------- | ---------------------------------- | --------------------- | ---------------------- |
| `[INSTANCE_ID].EntityDetailsCard.title`                   | Card title                         | Card header           | -                      |
| `[INSTANCE_ID].EntityDetailsCard.attributes.[FIELD_NAME]` | Label for each displayed attribute | InformationCard label | `[FIELD_NAME]` dynamic |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "EntityDetailsCard": {
      "title": "User Information",
      "attributes": {
        "email": "Email",
        "username": "Username"
      }
    }
  }
}
```

## Entity Attribute Fields

**Files:**

- apps/catalog-ui/src/components/field/EntityAttributeTextField.vue
- apps/catalog-ui/src/components/field/EntityAttributeNumberField.vue
- apps/catalog-ui/src/components/field/EntityAttributeBooleanField.vue
- apps/catalog-ui/src/components/field/EntityAttributeDateField.vue

| Key pattern                                | Description                                           | Usage                  | Parameters |
| ------------------------------------------ | ----------------------------------------------------- | ---------------------- | ---------- |
| `[INSTANCE_ID].fields.[FIELD_NAME].label`  | Field label (optional)                                | Input/toggle label     | -          |
| `[INSTANCE_ID].fields.[FIELD_NAME].hint`   | Helper text (optional)                                | Input hint             | -          |
| `[INSTANCE_ID].fields.[FIELD_NAME].prefix` | Input prefix (optional)                               | Input prefix           | -          |
| `[INSTANCE_ID].fields.[FIELD_NAME].suffix` | Input suffix (optional)                               | Input suffix           | -          |
| `[INSTANCE_ID].fields.[FIELD_NAME].close`  | Close label for the date picker (only for date field) | Date picker close text | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "fields": {
      "[FIELD_NAME]": {
        "label": "Email Address",
        "hint": "Enter a valid email",
        "prefix": "",
        "suffix": ""
      }
    }
  }
}
```
