# Catalog UI - Translation Keys Documentation

This document lists the translation keys used by the catalog-ui shared components. Scopes are provided by the host application; ensure the keys below exist in your locale files.

> **📋 Status:** Keys reflect the current implementation of the catalog UI components and layout.

---

## BaseLayout

**File:** apps/catalog-ui/src/layouts/BaseLayout.vue

| Key                   | Description            | Usage                | Parameters |
| --------------------- | ---------------------- | -------------------- | ---------- |
| `application.title`   | Application title text | Header toolbar title | -          |
| `application.version` | Version badge text     | Header badge label   | -          |

**Example:**

```json
{
  "application": {
    "title": "LinID Identity Manager",
    "version": "v1.0.0"
  }
}
```

---

## HeaderProfile

**File:** apps/catalog-ui/src/components/profile/HeaderProfile.vue

This component has no translation keys. User information (name and email) is read directly from the LinID user store.

## ButtonsCard

**File:** apps/catalog-ui/src/components/card/ButtonsCard.vue

| Key                                        | Description                  | Usage                 | Parameters |
| ------------------------------------------ | ---------------------------- | --------------------- | ---------- |
| `[INSTANCE_ID].ButtonsCard.title`          | Section title (optional)     | Card header           | -          |
| `[INSTANCE_ID].ButtonsCard.cancel`         | Cancel button label          | Cancel button text    | -          |
| `[INSTANCE_ID].ButtonsCard.confirm`        | Confirm button label         | Confirm button text   | -          |
| `[INSTANCE_ID].ButtonsCard.confirmLoading` | Confirm button loading label | Confirm button loader | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "ButtonsCard": {
      "title": "Actions",
      "cancel": "Cancel",
      "confirm": "Save",
      "confirmLoading": "Saving..."
    }
  }
}
```

## DropdownButton

**File:** apps/catalog-ui/src/components/button/DropdownButton.vue

| Key                                                   | Description                                | Usage                     | Parameters                          |
| ----------------------------------------------------- | ------------------------------------------ | ------------------------- | ----------------------------------- |
| `[INSTANCE_ID].DropdownButton.title`                  | Dropdown button label                      | Main button label         | -                                   |
| `[INSTANCE_ID].DropdownButton.[ITEM_KEY]`             | Label for a root item without children     | Direct root action label  | `[ITEM_KEY]` dynamic                |
| `[INSTANCE_ID].DropdownButton.[ITEM_KEY].title`       | Label for a root item that opens a submenu | Parent menu item label    | `[ITEM_KEY]` dynamic                |
| `[INSTANCE_ID].DropdownButton.[ITEM_KEY].[CHILD_KEY]` | Label for a child action inside a submenu  | Nested child action label | `[ITEM_KEY]`, `[CHILD_KEY]` dynamic |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "DropdownButton": {
      "title": "Actions",
      "edit": "Edit user",
      "archive": "Archive user",
      "export": {
        "title": "Export",
        "csv": "Export as CSV",
        "pdf": "Export as PDF"
      }
    }
  }
}
```

## AdvancedSearchCard

**File:** apps/catalog-ui/src/components/card/AdvancedSearchCard.vue

| Key                                                                                        | Description                                                             | Usage                      | Parameters |
| ------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------- | -------------------------- | ---------- |
| `[INSTANCE_ID].AdvancedSearchCard.title`                                                   | Card title (optional)                                                   | Card header                | -          |
| `[INSTANCE_ID].AdvancedSearchCard.moreFilters`                                             | Label to expand advanced filters                                        | Toggle button label        | -          |
| `[INSTANCE_ID].AdvancedSearchCard.lessFilters`                                             | Label to collapse advanced filters                                      | Toggle button label        | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].label`                               | Field label (optional)                                                  | Input/toggle label         | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].hint`                                | Helper text (optional)                                                  | Input hint                 | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].prefix`                              | Input prefix (optional)                                                 | Input prefix               | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].suffix`                              | Input suffix (optional)                                                 | Input suffix               | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].close`                               | Close label for the date picker                                         | Date picker close text     | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].validation.dynamicList.missingRoute` | Validation message when a Dynamic List field is missing its data route  | Dynamic List options error | -          |
| `[INSTANCE_ID].AdvancedSearchCard.fields.[FIELD_NAME].validation.dynamicList.fetchError`   | Validation message when fetching options for a Dynamic List field fails | Dynamic List options error | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "AdvancedSearchCard": {
      "title": "Filters",
      "moreFilters": "More filters",
      "lessFilters": "Fewer filters",
      "fields": {
        "[FIELD_NAME]": {
          "label": "Email Address",
          "hint": "Enter a valid email",
          "prefix": "",
          "suffix": ""
        }
      }
    }
  }
}
```

## GenericTree

**File:** apps/catalog-ui/src/components/tree/GenericTree.vue

| Key                                            | Description                                   | Usage                   | Parameters |
| ---------------------------------------------- | --------------------------------------------- | ----------------------- | ---------- |
| `[INSTANCE_ID].GenericTree.noNodesLabel`       | Message shown when the tree has no nodes      | Empty tree state        | -          |
| `[INSTANCE_ID].GenericTree.noResultsLabel`     | Message shown when a filter yields no results | Filtered empty state    | -          |
| `[INSTANCE_ID].GenericTree.types.[TYPE].label` | Display label for a given node type           | Node header in the tree | `value`    |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "GenericTree": {
      "noNodesLabel": "No items to display",
      "noResultsLabel": "No results",
      "types": {
        "[TYPE]": {
          "label": "{value}"
        }
      }
    }
  }
}
```
