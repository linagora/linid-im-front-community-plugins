## Navigation Menu

Translation keys for the module's navigation menu entry.

**File:** `apps/module-users/src/module-lifecycle.ts`

| Key                                  | Description                                 | Usage                     | Parameters |
| ------------------------------------ | ------------------------------------------- | ------------------------- | ---------- |
| `[INSTANCE_ID].NavigationMenu.label` | Label displayed in the main navigation menu | Navigation menu item text | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "NavigationMenu": {
      "label": "Users"
    }
  }
}
```

## HomePage

**File:** `apps/module-users/src/pages/HomePage.vue`

| Key                                | Description                              | Usage              | Parameters |
| ---------------------------------- | ---------------------------------------- | ------------------ | ---------- |
| `[INSTANCE_ID].HomePage.title`     | Page title for the user management list  | Page header        | -          |
| `[INSTANCE_ID].HomePage.error`     | Error message if user list fails to load | Error state        | -          |
| `[INSTANCE_ID].HomePage.seeButton` | Label for button to view user details    | Action button text | -          |

**Example:**

```json
{
  "[INSTANCE_ID]": {
    "HomePage": {
      "title": "Users Management",
      "error": "Failed to load users...",
      "seeButton": "See user"
    }
  }
}
```

**File:** `apps/module-users/src/pages/UserDetailsPage.vue`

| Key                                                                       | Description                                | Usage            | Parameters |
| ------------------------------------------------------------------------- | ------------------------------------------ | ---------------- | ---------- |
| `[INSTANCE_ID].UserDetailsPage.title`                                     | Page title prefix for viewing a user       | Page header      | -          |
| `[INSTANCE_ID].UserDetailsPage.edit`                                      | Label for edit button                      | Edit button text | -          |
| `[INSTANCE_ID].UserDetailsPage.error`                                     | Error message if user details fail to load | Error state      | -          |
| `[INSTANCE_ID].UserDetailsPage.ButtonsCard.cancel`                        | Label for back/cancel button               | Back button text | -          |
| `[INSTANCE_ID].UserDetailsPage.EntityDetailsCard.title`                   | Title for user information card            | Card header      | -          |
| `[INSTANCE_ID].UserDetailsPage.EntityDetailsCard.attributes.[FIELD_NAME]` | Label for user attribute field             | Field label      | -          |

**Example:**

```json
{

      "title": "User Details:",

      "error": "Failed to load user data",
      "ButtonsCard": {
        "cancel": "Back to Users List"
      },

          "lastName": "Last Name",
          "createdAt": "Created At",
```
